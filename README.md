# AtlasGuard

This is a simple website for AtlasGuard, a fictional company that provides cybersecurity solutions for travelers.

## Deployment to GitHub Pages

1.  Make sure you have a GitHub account.
2.  Create a new repository on GitHub.
3.  Push the files in this directory to the new repository.
4.  In the repository settings, go to the "Pages" section.
5.  Select the `main` branch as the source and click "Save".
6.  Your website will be available at `https://<your-username>.github.io/<repository-name>/`.
